# AI Memory Telegram Bot

Телеграм-бот на `aiogram 3`, который:

- держит краткосрочную память диалога на последние `20` сообщений;
- сохраняет текст и PDF через `/store` в ChromaDB;
- использует OpenAI для ответов и эмбеддингов;
- работает асинхронно и настраивается через `.env`.

## Возможности

- `/start` показывает короткую справку;
- `/store <текст>` сохраняет текст в долговременную память;
- `/store` с PDF-файлом в подписи к документу сохраняет и индексирует PDF;
- обычные текстовые сообщения идут в LLM с учетом последних сообщений и найденных релевантных фрагментов из памяти.

## Установка

1. Создайте виртуальное окружение и установите зависимости:

```bash
pip install -r requirements.txt
```

2. Скопируйте `.env.example` в `.env` и заполните значения:

```bash
copy .env.example .env
```

3. Укажите:

- `TELEGRAM_BOT_TOKEN`
- `OPENAI_API_KEY`

4. При необходимости поменяйте модель:

- `OPENAI_MODEL=gpt-5.4-nano`
- `OPENAI_EMBEDDING_MODEL=text-embedding-3-small`

## Запуск

```bash
python -m src.main
```

## Примеры

### Сохранить текст

```text
/store Меня зовут Алексей, я работаю с Python и Telegram-ботами.
```

### Сохранить PDF

Отправьте PDF-файл с подписью:

```text
/store
```

### Обычный диалог

Просто отправьте сообщение боту, и он ответит, учитывая:

- последние сообщения диалога;
- релевантные записи из ChromaDB.

## Структура

- `src/main.py` - точка входа и обработчики Telegram;
- `src/config.py` - загрузка настроек из `.env`;
- `src/openai_service.py` - запросы к OpenAI;
- `src/memory.py` - краткосрочная и долговременная память;
- `src/content.py` - разбиение текста и извлечение PDF;
- `requirements.txt` - зависимости.

## Примечание

Модель `gpt-5.4-nano` можно заменить на любую доступную в вашем аккаунте OpenAI.

Для этой среды я добавил локальный `chromadb`-совместимый слой, потому что официальный
Windows-wheel для `chroma-hnswlib` не ставится на Python 3.13 без C++ build tools.
Логика долговременной памяти при этом остается такой же: эмбеддинги сохраняются и
ищутся по косинусной близости.
